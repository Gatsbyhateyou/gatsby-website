# PRD 文档目录

本文件夹用于存放 Lin's lab 及相关项目的产品需求文档（PRD）。

## 命名规范

- **主文档**：统一使用 `PRD-001.md`、`PRD-002.md`、`PRD-003.md`…… 格式，按顺序递增。
- **文件名仅使用英文、数字、连字符**，禁止中文命名，以避免编码与跨平台问题。
- 同一 PRD 的对比、修订说明等可使用后缀，如 `PRD-001-compare.md`。

## 当前文件

| 文件 | 说明 |
|------|------|
| **PRD-001.md** | 当前版本产品需求文档（Lin's lab，基于线上站点形态） |
| **PRD-001-compare.md** | 早期交接/记忆摘要与 PRD-001 的并列对比 |
| **PRD-002-ui-style-prompt-gallery.md** | UI 风格提示词画廊：17 种风格、CSS 小样与预览页、关键词/草稿/skill-creator/接入脚本流程 |

## 后续

- 新 PRD 主文档命名为 `PRD-002.md`、`PRD-003.md`…… 依次递增。
- 对比、修订、附录等可放在本目录，文件名保持英文（如 `PRD-001-compare.md`、`PRD-002-changelog.md`）。
