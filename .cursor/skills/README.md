# Cursor 技能（由「个人主页+AI工具导航」迁移）

本目录下的技能自项目「个人主页+AI工具导航」的 `.cursor/skills/` 迁移而来，原由云舒 yunshu_skillshub 转换为 Cursor Agent Skills 格式，便于在本仓库中直接使用。

## 技能列表

| 技能名 | 用途 | 触发示例 |
|--------|------|----------|
| version-planner | 版本规划（V0.1 MVP→V1.0） | 拆版本、版本规划、MVP怎么做 |
| project-map-builder | 生成/更新 PROJECT_MAP.md | 目录地图、项目地图、更新 PROJECT_MAP |
| req-change-workflow | 需求变更标准化流程 | 改需求、需求变更、调整交互 |
| thinking-partner | 思考拍档，理清局面与卡点 | 很乱帮我理一下、陪我想想 |
| thought-mining | 思维挖掘→成稿 | 我想写关于XX的文章、整理想法 |
| writing-assistant | 写作助手（选题→框架→内容） | 我想写XX、梳理选题、组织思路 |
| weekly-report | 周报梳理 | 写周报、梳理周报、整理工作 |
| priority-judge | 优先级判断 | 很多事要做、排优先级、今天该做什么 |
| prd-doc-writer | PRD/需求文档撰写 | 写PRD、梳理需求文档、用户故事 |
| lesson-builder | 课程大纲与课件 | 备课、做课件、准备课程 |
| image-assistant | 配图与出图提示词 | 配图、出图提示词、字太多更好读 |
| ui-design | UI 样式修改协作 | 改页面样式、调整布局、改间距颜色 |
| find-skills | 在 skills.sh 生态中搜索/安装技能 | 找技能、有没有做X的技能、npx skills、想扩展 Agent 能力 |
| web-design-guidelines | 按 Vercel Web Interface Guidelines 审查 UI | 检查UI、审查设计、检查无障碍、web design guidelines |
| frontend-design | 有辨识度的前端界面设计（避免 AI 味） | 做前端、做页面、美化界面、frontend design、做落地页 |
| skill-creator | 创建/更新 Agent Skill 的指南（Anthropic） | 创建 skill、写新技能、更新 SKILL.md、skill 格式 |
| cleanup-unused | 版本大更新后清理未引用文件与死代码/注释块 | 清理、跑一遍清理检查、删掉没用的、清理死代码 |

## 来源
