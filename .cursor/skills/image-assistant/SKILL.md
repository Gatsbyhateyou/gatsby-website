# 配图助手

五阶段顺序执行，每阶段告知用户当前阶段与输出物，并读取对应 stage 文件执行。

## 阶段与目标

| 阶段 | 名称       | 目标                         | 详细文件 |
|------|------------|------------------------------|----------|
| 1    | 需求澄清   | 内容/场景/受众/字多字少；需求小结 | stages/01-brief.md |
| 2    | 配图规划   | 拆内容→图清单（几张、每张讲啥、模板） | stages/02-plan.md |
| 3    | 文案定稿   | 逐字定稿「图上写什么」(Copy Spec) | stages/03-copy.md |
| 4    | 提示词封装 | 封装为可复制提示词；可选批量出图 | stages/04-prompts.md |
| 5    | 迭代润色   | 按反馈减字、换隐喻、提可读   | stages/05-iterate.md |

## 调度

- 需求未锁（内容/场景/受众/字多字少）→ 阶段1。
- 需拆块或定「几张图/每张讲什么」→ 阶段2。
- 图清单已有，未定图上逐字文案 → 阶段3。
- Copy Spec 已确认，要出提示词/批量出图 → 阶段4。
- 用户反馈字多/不好看 → 阶段5（必要时回阶段1）。

## 规范

每张图一个核心信息；中文少字短句、大字号；每张提示词单独代码块；默认 16:9；Copy Spec 确认后阶段4 不擅自改文案，只做封装与参数补全。
