# 可爱风 Cute / Kawaii UI

做界面时：**柔和 Pastel 色**（粉、薄荷、薰衣草、奶油）、**大圆角**（border-radius 高）、**软阴影**（无硬边）、可加俏皮元素（小圆点、心形、星星、气泡）。整体温暖、治愈、偏少女/萌系。与「有机」的区别是更强调「可爱」气质与 Pastel 色；与「极简」的区别是更饱满、更有装饰感。

## 何时用

- 面向年轻女性或萌系受众的产品
- 手作/文具/宠物/亲子/生活方式类
- 需要温暖、亲和、无攻击性的品牌气质

## Prompt（供画廊复制）

请用可爱/Kawaii 风格做界面：柔和 Pastel 色（粉、薄荷、薰衣草、奶油、 Peach 等）、大圆角（border-radius 16px 以上）、软阴影（blur 明显、无硬边）、背景以浅色为主。可加俏皮元素（小圆点、心形、星星、气泡）。字体可选圆润款，按钮/卡片 hover 时有轻微弹跳或缩放。整体温暖、治愈、萌系；避免冷峻、硬朗、科技感强的视觉。
