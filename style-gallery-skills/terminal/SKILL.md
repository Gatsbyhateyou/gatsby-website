# 终端/CLI 风 Terminal UI

做界面时：等宽字体（monospace）、深色背景（如 #0d1117、#1e1e1e）、高对比文字（绿/黄/灰）；可选边框与「命令行」式区块，按钮/标签可做成类 CLI 的简洁样式。

## Prompt（供画廊复制）

请用终端/CLI 风格做界面：等宽字体、深色背景（如 #0d1117）、高对比文字（绿/黄/灰/白）；区块与按钮简洁、类命令行，可选细边框与「提示符」式装饰。整体像开发工具或终端仪表盘，信息密度适中、可读性强。
