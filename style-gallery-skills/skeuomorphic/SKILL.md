# 拟物化风格 Skeuomorphic UI

做界面时：用质感与阴影模拟真实物体——皮革、木纹、金属、布料、玻璃等；高光与暗部明确、有立体感；图标与控件像可触摸的实体。适合需要「像真实设备/实物」气质的界面（如早期 iOS、音乐/创意工具）。

## 何时用

- 用户希望界面有「实物感」「质感」
- 品牌或产品需要温暖、可触摸的视觉
- 需要与扁平/极简形成对比的强调区域

## Prompt（供画廊复制）

请用拟物化（skeuomorphic）风格做界面：用视觉隐喻模拟真实材质——皮革、木纹、金属、玻璃等；明确的高光与暗部、立体阴影与质感，图标与控件像可触摸的实体。避免纯扁平与大面积纯色，保持层次与「实物感」。适合仪表盘控件、音乐/创意类工具、或需要温暖质感的界面。
