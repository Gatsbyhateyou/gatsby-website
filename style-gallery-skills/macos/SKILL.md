# macOS 设计风格

做界面时：**毛玻璃与半透明**（vibrancy、backdrop-blur）、大圆角（10–12px）、SF Pro 气质的无衬线字体；浅色背景、活力强调色（系统蓝 #0071e3、绿、橙）。侧边栏与工具栏层级清晰，图标简洁可辨识。整体通透、克制、符合 Apple HIG。与「玻璃拟态」的区别是更偏浅色与系统色；与「iOS」的区别是桌面端布局与控件形态。

## 何时用

- 面向 Mac 用户的桌面端或 Web 应用
- 需要「原生 macOS 感」的工具或品牌
- 与 Apple 生态调性一致的产品

## Prompt（供画廊复制）

请用 macOS Big Sur / Monterey 设计风格做界面：毛玻璃与半透明（backdrop-blur、vibrancy）、大圆角（10–12px）、SF Pro 气质字体；浅色背景、活力强调色（系统蓝/绿/橙）。侧边栏与工具栏层级清晰，图标简洁。整体通透、克制、符合 Apple Human Interface Guidelines。
