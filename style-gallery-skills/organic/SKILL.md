# 有机/自然视觉风格 Organic UI

做界面时：**流动的曲线**与有机形态（非严格网格）、柔和色彩（自然色、渐变）、留白与呼吸感；可配合柔和动效（缓动、弹性）。整体像自然生长、非机械排版。与「极简」的区别是更偏曲线与有机形；与「拟物」的区别是不模拟具体材质，而是气质上的自然与流动。

## 何时用

- 品牌或产品强调自然、有机、可持续、生命感
- 需要与「科技/硬朗」形成对比的温暖界面
- 落地页、品牌页、艺术/生活方式类产品

## Prompt（供画廊复制）

请用有机/自然视觉风格做界面：流动的曲线与有机形态（非严格网格）、柔和色彩与自然感渐变、留白与呼吸感；可配合柔和动效。整体像自然生长、非机械排版；避免硬边与直角主导，保持温暖、流动的气质。适合品牌页、落地页或强调自然/生命感的产品。
