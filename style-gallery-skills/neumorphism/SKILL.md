# 新拟态风格 Neumorphism (Soft UI)

做界面时：背景与控件同色系（常见浅灰），用**双阴影**（一侧亮、一侧暗）营造凸起或凹陷；无硬边框或细边框，圆角较大；整体柔和、像从背景「长出来」。注意对比度与无障碍，避免在低对比背景下使用。

## 何时用

- 需要「软」「可触摸」的现代质感，又不想用拟物材质
- 品牌偏柔和、克制，与粗野/高对比风格相反
- 小范围使用（卡片、按钮），避免整页导致可读性下降

## Prompt（供画廊复制）

请用新拟态（neumorphism / soft UI）风格做界面：背景与控件同色系（如浅灰 #e0e0e0），用双阴影（一侧亮、一侧暗）营造凸起或凹陷，无硬边框、大圆角；整体柔和、像从背景浮起或压入。注意对比度与可读性，避免大面积使用导致层次不清。
