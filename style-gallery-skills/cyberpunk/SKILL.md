# 赛博朋克风格 Cyberpunk UI

做界面时：**霓虹色**（黄 #FCEe0a、粉 #FF2A6D、青 #05d9e8）、故障效果（glitch、色偏、扫描线）、高对比暗背景（深黑 #0d0d0f）；科技感/等宽字体。边框光效、网格辅助线、HUD 式边角。整体赛博、未来、反乌托邦气质。参考赛博朋克 2077 的黄/粉/青配色与 HUD 感。与「暗黑主题」的区别是强调霓虹与故障；与「终端风」的区别是更偏视觉冲击而非命令行。

## 何时用

- 游戏、科幻、音乐、潮流品牌
- 需要强视觉冲击与科技感的落地页
- 与「温暖/自然」形成反差的产品

## Prompt（供画廊复制）

请用赛博朋克/夜城风格做界面：霓虹色密布（黄 #FCEe0a、粉 #FF2A6D、青 #05d9e8）、高对比暗背景（#0d0d0f）；科技感/等宽字体。可加：雨雾氛围（雨丝、雾蒙蒙边缘）、霓虹光晕缓慢浮动、广告牌式发光字、扫描线/网格、底部反射光带。边框光效、HUD 式边角。整体拥挤、层次多、夜城感——霓虹密布、潮湿、热闹。参考赛博朋克 2077。避免柔和、自然、萌系。
