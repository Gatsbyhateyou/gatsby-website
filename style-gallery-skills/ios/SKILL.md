# iOS 设计风格

做界面时：**大标题**（large title，2rem+）、毛玻璃导航与 tab 栏；圆角卡片（16px）、安全区（env(safe-area-inset)）、触控友好（最小 44pt）。SF Pro 字体、系统色（蓝 #007aff、绿、橙、红）。列表与分组、分段控件、操作表。整体符合 iOS HIG，原生感强。与「macOS」的区别是移动端布局与触控优先；与「极简」的区别是有明确的系统控件形态。

## 何时用

- 面向 iPhone / iPad 的 Web 应用
- 需要「像 App」的移动端体验
- PWA 或混合应用界面

## Prompt（供画廊复制）

请用 iOS / Apple 移动端设计风格做界面：大标题（large title）、毛玻璃导航与 tab 栏；圆角卡片、安全区、触控友好（最小 44pt）；SF Pro 气质、系统色。列表与分组、分段控件、操作表。整体符合 iOS Human Interface Guidelines，原生感强。
