# 卡片式布局 Card Layout UI

做界面时：**信息以卡片为单位**，每张卡片有标题、摘要、可选图；可网格或瀑布流排列。卡片有统一圆角（10px）与阴影、hover 有反馈。适合内容聚合、商品展示、社交媒体流。整体清晰、易扫读、模块化。与「Bento」的区别是卡片大小更统一、偏流式；与「干净界面」的区别是更强调卡片这一信息载体。

## 何时用

- 内容平台、电商、社交类产品
- 需要卡片式信息架构的界面
- 类似 Pinterest / Dribbble 的流式布局

## Prompt（供画廊复制）

请用卡片式布局风格做界面：信息以卡片为单位，每张卡片有标题、摘要、可选图；网格或瀑布流排列。卡片统一圆角与阴影、hover 有反馈。适合内容聚合、商品展示。整体清晰、易扫读、模块化，像 Pinterest / Dribbble 的流式布局。
