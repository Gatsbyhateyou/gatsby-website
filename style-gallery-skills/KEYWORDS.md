# UI 风格 · 可搜关键词表

搜技能时用**英文关键词**（`npx skills find <关键词>` 或 `.\scripts\fetch-top-skill-to-draft.ps1 <关键词>`）。下表按视觉气质、平台、年代分类，供选风格与写 prompt 时参考。

---

## 一、按视觉气质分

| 风格（中/英） | 简要说明 | 可搜关键词示例 |
|---------------|----------|----------------|
| 极简 / Minimal | 大量留白、少装饰、字与间距为主 | minimal, minimalism, clean ui |
| 极繁 / Maximalist | 装饰多、色彩/图案/层次丰富 | maximalist, bold, expressive |
| 新拟态 / Neumorphism | 柔和的凸起/凹陷、像真实物体 | neumorphism, soft ui |
| 玻璃拟态 / Glassmorphism | 毛玻璃、半透明、模糊背景 | glassmorphism, glass, frosted |
| 扁平 / Flat | 无立体、纯色块、少渐变与阴影 | flat design, flat ui |
| 拟物 / Skeuomorphic | 模仿真实材质（皮革、金属等） | skeuomorphic, realistic ui |
| 复古 / Retro | 老式排版、旧广告/杂志感 | retro, vintage, 80s, 90s |
| 赛博/科技 / Cyber, Tech | 深色、霓虹、网格、科技感 | cyber, tech, neon, hacker |
| 有机/自然 / Organic | 曲线、自然色、手绘/植物感 | organic, nature, soft, rounded |
| 工业/实用 / Industrial | 粗体、高对比、偏工具感 | industrial, utility, brutalist |
| 杂志/编辑 / Editorial | 大标题、分栏、像杂志排版 | editorial, magazine, typography |
| 暗黑 / Dark | 深色背景、高对比、护眼向 | dark mode, dark theme |
| 柔和/马卡龙 / Pastel | 淡色、低饱和、柔和 | pastel, soft colors, candy |
| 高对比/高可读 / High contrast | 强对比、无障碍、可读优先 | high contrast, accessibility |
