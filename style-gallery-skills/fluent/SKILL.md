# Microsoft Fluent Design 风格

做界面时：**亚克力材质**（acrylic：半透明 + backdrop-blur）、深度与层次（elevation、reveal 高亮）、柔和光影；圆角适中（4–8px）。微动效（hover 高亮、press 反馈）。色彩可跟随 Fluent 色板（中性灰 #f3f2f1、品牌蓝 #0078d4）。整体现代、专业、企业级。与「玻璃拟态」的区别是更偏企业感与微软控件形态；与「干净界面」的区别是有更多材质与层次。

## 何时用

- 面向 Windows 或企业用户的产品
- 需要与 Microsoft 生态一致的工具
- 企业后台、办公、协作类应用

## Prompt（供画廊复制）

请用 Microsoft Fluent Design 2 风格做界面：亚克力材质（半透明 + backdrop-blur）、深度与层次（elevation、reveal 效果）、柔和光影；圆角 4–8px。微动效（hover 高亮、press 反馈）。色彩可用中性灰 + 品牌强调色。整体现代、专业、企业级，符合 Fluent 设计语言。
