# 干净界面 Clean UI

做界面时：**克制的配色**（黑白灰 #0f172a / #64748b + 单一强调色）、清晰的层次（字号/字重/间距）；少装饰、无多余边框与阴影。高可读、高对比、留白充足。整体干净、专业、像 Stripe / Linear / Vercel 的克制感。与「极简」的区别是仍保留必要的层次与控件；与「界面设计」的区别是更强调「干净」气质而非流程设计。

## 何时用

- 后台、工具、SaaS 产品
- 需要专业、可信赖气质的界面
- 信息密度适中、不抢眼的 UI

## Prompt（供画廊复制）

请用干净界面风格做界面：克制的配色（黑白灰 + 单一强调色）、清晰层次、少装饰；高可读、高对比、留白充足。避免花哨边框与阴影。整体干净、专业、像 Stripe / Linear / Vercel 的克制感。
